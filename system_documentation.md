## 1. نظرة عامة على المشروع

### 1.1 ما هو AMAN ERP؟
نظام تخطيط موارد مؤسسات (ERP) شامل ومتعدد المستأجرين، مصمم خصيصاً للسوق العربي مع امتثال كامل للأنظمة السعودية (ZATCA، GOSI، WPS، الزكاة). يغطي **16 وحدة متكاملة** في منصة واحدة.

### 1.2 أرقام النظام
| المقياس | القيمة |
|---------|--------|
| نقاط API | 767 (392 GET + 244 POST + 79 PUT + 52 DELETE) |
| صفحات الواجهة | 277 صفحة JSX / ~270 مسار |
| جداول قاعدة البيانات | 244 جدول لكل شركة |
| قيود يومية تلقائية | 75+ نقطة تفعيل |
| قوالب قطاعات | 12 قالب |
| أدوار افتراضية | 8 أدوار |
| أسطر كود Backend | 88,268 سطر Python |
| أسطر كود Frontend | 98,760 سطر React JSX |

### 1.3 التقنيات المستخدمة
| الطبقة | التقنية |
|--------|---------|
| Backend | Python (FastAPI) |
| Frontend | React JSX (Vite) |
| قاعدة البيانات | PostgreSQL (قاعدة بيانات لكل مستأجر) |
| التخزين المؤقت | Redis |
| الحاويات | Docker + Docker Compose |
| خادم الويب | Nginx (Reverse Proxy) |
| المصادقة | JWT (Bearer Token) |

### 2.2 نموذج تعدد المستأجرين (Multi-Tenancy)
- كل شركة تحصل على قاعدة بيانات PostgreSQL مستقلة: `aman_{company_id}`
- قاعدة بيانات فهرس على مستوى النظام (`postgres`) تخزن سجل الشركات وتوجيه المستخدمين
- تسجيل الدخول يستخدم جدول `system_user_index` للبحث السريع

## 3. المصادقة والأمان

### 3.1 صفحات المصادقة

| الصفحة | المسار | الوصف |
|--------|--------|-------|
| تسجيل الدخول | `/login` | إدخال اسم المستخدم وكلمة المرور ورمز الشركة |
| التسجيل | `/register` | إنشاء حساب شركة جديدة |
| نسيت كلمة المرور | `/forgot-password` | طلب رابط إعادة تعيين |
| إعادة تعيين كلمة المرور | `/reset-password` | تعيين كلمة مرور جديدة |

### 3.2 آلية العمل
1. المستخدم يرسل `POST /api/auth/login` بـ `username` + `password` + `company_code` (form-data)
2. النظام يتحقق ويعيد `access_token` (JWT صالح 30 دقيقة)
3. جميع الطلبات اللاحقة تحتاج `Authorization: Bearer <token>`
4. تحديد المعدل: 10 طلبات/دقيقة، 5 طلبات/IP، حظر 15 دقيقة
5. دعم المصادقة الثنائية (2FA) عبر TOTP

### 3.3 الأدوار والصلاحيات
| الدور | بالعربي | مستوى الوصول |
|-------|---------|-------------|
| `superuser` | مدير النظام | وصول كامل `["*"]` |
| `admin` | مدير | وصول كامل |
| `manager` | مدير فرع | مبيعات، مشتريات، مخزون، نقاط بيع، تقارير |
| `accountant` | محاسب | محاسبة، خزينة، ضرائب، تقارير |
| `sales` | مبيعات | مبيعات، منتجات، نقاط بيع، عقود |
| `inventory` | أمين مستودع | مخزون، منتجات، تصنيع (عرض فقط) |
| `cashier` | كاشير | نقاط البيع، مبيعات (عرض فقط) |
| `user` | مستخدم | لوحة التحكم فقط |

**صيغة الصلاحية:** `module.action` (مثال: `sales.create`, `accounting.manage`)
