## الملفات المعنية

### Backend
- industry_coa_templates.py — قوالب شجرة الحسابات
- industry_gl_rules.py — قواعد الترحيل التلقائي
- industry_kpi_service.py — مؤشرات الأداء
- main.py — بيانات industry_templates (12 صف)
- companies.py — زرع COA عند إنشاء شركة
- settings.py — زرع COA عند تغيير نوع النشاط
- dashboard.py — endpoints: industry-widgets, gl-rules, coa-summary
- accounting.py — _ACCOUNT_MODULE_MAP

### Frontend
- industryModules.js — تعريف 12 نشاط + الوحدات المتغيرة
- IndustrySetup.jsx — معالج اختيار النشاط (12 بطاقة)
- IndustryWidgets.jsx — ويدجتات حسب النشاط
- ReportCenter.jsx — تقارير متخصصة حسب النشاط
- useIndustryType.js — هوك إدارة نوع النشاط
- Sidebar.jsx — تصفية القائمة الجانبية حسب الوحدات
