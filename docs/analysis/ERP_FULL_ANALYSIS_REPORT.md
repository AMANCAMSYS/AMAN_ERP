## القسم الأول: آلية ربط شجرة الحسابات بالنظام

### 1.1 نظرة عامة على آلية الربط

يعتمد نظام أمان ERP على آلية ربط ذكية ومتماسكة بين العمليات التجارية المختلفة وشجرة الحسابات المحاسبية. هذه الآلية تعمل من خلال **دالة مركزية رئيسية** تُدعى `get_mapped_account_id` موجودة في ملف `backend/utils/accounting.py`.

تعمل هذه الدالة كمحور رئيسي لاسترجاع معرفات الحسابات المحاسبية المرتبطة بوظائف محددة في النظام، مما يضمن أن كل عملية تجارية يتم تسجيلها تؤثر بشكل صحيح على الحسابات المحاسبية المناسبة.

### 1.2 مبدأ عمل مفاتيح الربط (Mapping Keys)

تعمل آلية الربط على مبدأ التخزين في جدول `company_settings`:

```
┌─────────────────────────────────────────────────────────┐
│                    company_settings                      │
├─────────────────────┬───────────────────────────────────┤
│  setting_key        │  setting_value (account_id)       │
├─────────────────────┼───────────────────────────────────┤
│  acc_map_ar         │  1101  (المدينون)                 │
│  acc_map_ap         │  2101  (الدائنون)                 │
│  acc_map_sales_rev  │  4101  (إيرادات المبيعات)         │
│  acc_map_inventory  │  1201  (المخزون)                  │
│  acc_map_vat_out    │  2301  (ضريبة مخرجات)             │
│  ...                │  ...                              │
└─────────────────────┴───────────────────────────────────┘
```

**النمط الموحد:** كل مفتاح يبدأ بـ `acc_map_` متبوعاً بوصف مختصر للحساب.

### 1.3 الجداول الرئيسية المستخدمة

| الجدول | الدور |
|--------|-------|
| `accounts` | جميع الحسابات المحاسبية (أرقام، أسماء، أنواع، أرصدة) |
| `company_settings` | مفاتيح الربط والقيم المقابلة (معرفات الحسابات) |
| `journal_entries` | القيود المحاسبية الفعلية (مدين/دائن) |
| `journal_entry_lines` | سطور القيد المحاسبي (حساب + مبلغ + اتجاه) |

## القسم الثاني: متطلبات الوصول إلى 100% لكل وحدة

### 2.1 وحدة التصنيع (Manufacturing) — من 40% إلى 100%

**المشكلة الرئيسية:** عدم وجود ربط محاسبي كامل وعدم وجود حسابات محاسبية مُربطة.

#### ما هو موجود:
- قائمة المواد (BOM) مع إنشاء وتعديل وحذف
- أوامر الإنتاج مع حالات (معلق، قيد التنفيذ، مكتمل، ملغى)
- مراكز العمل الأساسية
- مسارات التصنيع (Routing)
- Job Cards أساسية

#### ما ينقص:

**الحسابات المحاسبية:**
- `acc_map_raw_materials` — تتبع تكلفة المواد الخام المستهلكة
- `acc_map_wip` — تكلفة الإنتاج غير المكتمل
- `acc_map_finished_goods` — تكلفة المنتجات المكتملة
- `acc_map_direct_labor` — تكاليف العمالة المباشرة
- `acc_map_mfg_overhead` — التكاليف غير المباشرة

**القيود التلقائية:**
1. **قيد إصدار أمر إنتاج:** مدين WIP ← دائن مواد خام (رمز `mo_issue`)
2. **قيد إنهاء أمر إنتاج:** مدين منتجات تامة ← دائن WIP (رمز `mo_complete`)
3. **قيد تكاليف العمالة:** مدين WIP ← دائن عمالة مباشرة
4. **قيد الأعباء الصناعية:** مدين WIP ← دائن أعباء صناعية

**الوظائف البرمجية:**
- دالة `calculate_production_cost` لحساب التكلفة الكلية
- BOM المتغيرة (Variable BOM) بنسب مئوية
- فحص جودة أثناء الإنتاج (In-Process QC)

**التقارير:**
- تكلفة الإنتاج الفعلي vs المقدر
- كفاءة الإنتاج وأداء مراكز العمل
- المواد المستهلكة والعمالة المباشرة والأعباء الصناعية

### 2.2 وحدة المشاريع (Projects) — من 35% إلى 100%

**المشكلة الرئيسية:** عدم وجود ربط محاسبي كامل وعدم وجود تقارير تفصيلية.

#### ما هو موجود:
- إدارة أساسية للمشاريع مع إنشاء ومتابعة المهام
- تسجيل التكاليف
- مخطط جانت (Gantt Chart) أساسي
- تتبع الوقت (Timesheet) أساسي

#### ما ينقص:

**الحسابات المحاسبية:**
- `acc_map_project_costs` — تتبع جميع مصروفات المشروع
- `acc_map_project_revenue` — تتبع إيرادات الفوترة
- `acc_map_billable_expenses` — المصروفات القابلة للفوترة على العميل

**القيود التلقائية:**
1. **قيد تسجيل تكلفة:** مدين تكاليف المشروع ← دائن المصروف المناسب
2. **قيد إصدار فاتورة:** مدين المدينون ← دائن إيرادات المشاريع + VAT
3. **قيد ربح/خسارة:** مقارنة الإيرادات بالتكاليف عند إغلاق المشروع

**الوظائف المفقودة:**
- Earned Value Management (EVM) — PV, EV, AC, SPI, CPI
- Change Orders (تغيير نطاق المشروع)
- أنواع عقود متعددة (Fixed Price, Time & Materials, Retainer)
- تنبيهات التأخر في الجدول

**التقارير:**
- تكلفة المشروع Actual vs Budget
- ربحية المشروع (Project Profitability)
- أداء الجدول واستخدام الموارد والانحراف

### 2.3 وحدة الضرائب (Taxes) — من 60% إلى 100%

**المشكلة الرئيسية:** عدم وجود دعم لـ ZATCA Phase 2 (إلزامي في المملكة العربية السعودية).

#### ما هو موجود:
- إنشاء إقرار ضريبي يحسب تلقائياً من فواتير المبيعات والشراء
- دعم الفترات الشهرية والربع سنوية
- حساب ضريبة المخرجات والمدخلات والمرتجعات
- قيد محاسبي تلقائي عند تسجيل الدفعة الضريبية
- قيد تسوية بين ضريبة المخرجات والمدخلات
- تقارير ضريبية تفصيلية

#### ما ينقص:

**ZATCA Phase 2:**
- API Client للتواصل مع خوادم ZATCA
- الحصول على شهادة CSID حقيقية
- التوقيع الرقمي (SHA-256 مع RSA)
- Hash فريد لكل فاتورة
- QR Code بصيغة TLV
- Simplified Tax Invoice (B2C) + Standard Tax Invoice (B2B)
- تصدير XML حسب مواصفات ZATCA

**ضرائب أخرى:**
- ضريبة الاستقطاع (WHT)
- إشعارات مواعيد التقديم الضريبي

### ما يتميز به نظام أمان ✅
- **Multi-Tenant حقيقي:** كل شركة = قاعدة بيانات PostgreSQL منفصلة (أفضل من ERPNext)
- **بحث مستخدم مُفهرس:** بحث سريع O(1)
- **Rate Limiting ثنائي:** حماية بالـ IP + اسم المستخدم
- **Permission Wildcards مرنة**
- **ربط محاسبي شامل** لمعظم العمليات
- **14 وحدة HR** أشمل من SAP B1

### ما ينقص نظام أمان ❌
- الفوترة الإلكترونية ZATCA (إلزامية في السعودية)
- تصدير PDF احترافي للفواتير
- Token Blacklist
- Webhook / Integration API
- تطبيق جوال
- CRM Module

## القسم الثامن: التقارير

### 8.1 التقارير الموجودة ✅

**تقارير المخزون:** ملخص المخزون، مخزون المستودع، حركة المخزون، تقييم المخزون، المخزون المنخفض، تقارير الدفعات

**تقارير المبيعات:** ملخص المبيعات، مبيعات حسب المنتج/العميل/الفئة، أفضل المنتجات، هوامش الربح

**تقارير المشتريات:** ملخص المشتريات، مشتريات حسب المورد/المنتج، مقارنة الموردين

**تقارير المحاسبة:** ميزان المراجعة، دفتر الأستاذ العام، قائمة الدخل، الميزانية العمومية، التدفقات النقدية

**تقارير الخزينة:** تقرير الأرصدة والتدفقات

**تقارير HR:** تقرير الحضور والإجازات والرواتب وتكلفة العمالة

**تقارير الضرائب:** تقرير VAT، تقرير التدقيق، ملخص الضرائب

### 8.2 التقارير المفقودة ❌

**تقارير مالية:** التدفقات النقدية التفاعلي، المقارنة بين الفترات، التحليل الأفقي، النسب المالية، مركز التكلفة، P&L تفصيلي

**تقارير مخزون:** تقييم بالتكلفة المتوسطة، دوران المخزون، المخزون الراكد، COGS

**تقارير مبيعات:** أداء البائعين، عمولات المبيعات، المبيعات المستهدفة

**تقارير تصنيع:** تكلفة الإنتاج الفعلية، كفاءة الإنتاج، المواد المستهلكة، العمالة، الأعباء

**تقارير مشاريع:** تكلفة المشروع، ربحية المشروع، تقدم المشروع، استخدام الموارد، الانحراف

**مشاكل التقارير الحالية:** عدم وجود تصدير PDF/Excel احترافي، رسوم بيانية تفاعلية محدودة
