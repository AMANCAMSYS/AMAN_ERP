## 🖱️ 1. أزرار ورموز النظام العامة
هذه الأزرار تظهر في معظم الصفحات، وإليك وظيفة كل منها بالتفصيل:

| الزر / الرمز | الاسم | الوظيفة والشرح |
| :--- | :--- | :--- |
| <span style="background:#0d6efd; color:white; padding:2px 5px; border-radius:4px;">+ جديد</span> | **إنشاء جديد** | يفتح صفحة فارغة لإدخال بيانات سجل جديد. |
| <span style="background:#198754; color:white; padding:2px 5px; border-radius:4px;">حفظ</span> | **حفظ** | يحفظ البيانات كـ "مسودة" (Draft). |
| <span style="background:#ffc107; color:black; padding:2px 5px; border-radius:4px;">تعديل</span> | **تعديل** | يفتح السجل الحالي للتعديل. |
| <span style="background:#dc3545; color:white; padding:2px 5px; border-radius:4px;">حذف</span> | **حذف** | يحذف السجل نهائياً (إذا لم يكن مرتبطاً بعمليات أخرى). |
| <span style="background:#0dcaf0; color:black; padding:2px 5px; border-radius:4px;">اعتماد</span> | **Submit / Approve** | يحول السجل إلى "نهائي" ويولد القيود المحاسبية. |
| 🖨️ | **طباعة** | يولد ملف PDF رسمي للمستند. |
| 📊 | **تصدير Excel** | يحمل بيانات الجدول كملف Excel. |

## 💰 2. المبيعات (Sales)
**المسار الرئيسي:** `/sales`

| العملية | المسار في المتصفح (URL) | شرح الخطوات |
| :--- | :--- | :--- |
| **قائمة العملاء** | `/sales/customers` | عرض جميع العملاء والبحث عنهم. |
| **إضافة عميل جديد** | `/sales/customers/new` | إدخال بيانات عميل جديد وحد ائتمانه. |
| **فاتورة مبيعات** | `/sales/invoices` | عرض قائمة جميع الفواتير الصادرة. |
| **إنشاء فاتورة جديدة** | `/sales/invoices/new` | اختيار العميل والأصناف والضريبة ثم الاعتماد. |
| **عروض الأسعار** | `/sales/quotations` | إدارة العروض المرسلة للعملاء قبل تأكيدها. |
| **أوامر البيع** | `/sales/orders` | تأكيد طلبات العملاء قبل الفوترة. |
| **مردودات المبيعات** | `/sales/returns` | تسجيل البضائع المعادة من العملاء. |
| **سندات القبض** | `/sales/receipts` | تسجيل استلام المبالغ النقدية أو البنكية من العملاء. |
| **كشف حساب عميل** | `/sales/reports/customer-statement` | تقرير يوضح حركات ومديونية عميل محدد. |
