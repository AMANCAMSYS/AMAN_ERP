## 1. الأساس الشرعي — Sharia Basis

| المصدر | النص |

## 2. طرق الحساب المدعومة — Supported Methods

### 🟢 الطريقة الأولى (الافتراضية): الموجودات الشرعية — Sharia Assets Method

> **هذه هي الطريقة الصحيحة شرعياً** — تعتمد جانب الأصول (الموجودات) فقط من قائمة المركز المالي

#### الخطوات العشر:

**الخطوة 1:** تعتمد قائمة المركز المالي (الميزانية) فقط — تستبعد القوائم الأخرى (الدخل، التدفقات النقدية، التغيرات في حقوق الملكية).

**الخطوة 2:** يُستبعد جانب المطلوبات بالكامل (حقوق الملكية + الدائنون) — لأنها حقوق مجردة وليست حقائق فعلية، والوجود الفعلي يظهر في الموجودات.

**الخطوة 3:** يُعتمد جانب الموجودات (الأصول) فقط — لأنه المحل الذي تدخله الزكاة في الإسلام.

**الخطوة 4:** تُصنف جميع بنود الموجودات وفقاً للأصول المالية الستة:
- النقد ✅
- عروض التجارة ✅
- الاستثمار ❌
- الإجارة ❌
- المدينون ❌
- الاستهلاك ❌

**الخطوة 5:** يُعتمد أصل **النقد** — بالإجماع (الذهب والفضة والنقود والأرصدة البنكية).

**الخطوة 6:** يُعتمد أصل **عروض التجارة** — كل مال أُعد للبيع في سوقه خلال السنة (المخزون).

**الخطوة 7:** تُستبعد بقية الأصول الأربعة:
- **الاستثمارات**: ملك ناقص (أوعية استثمارية لا يملكها الشركة ملكاً تاماً)
- **الأصول المؤجرة**: الزكاة في إيراداتها فقط (وهي تدخل النقدية تلقائياً)
- **المدينون**: ملك ناقص (المال بيد غيرك ولم يُقبض بعد)
- **المستهلكات**: أصول ثابتة + تحت الإنشاء + معنوية (ليست نقوداً وليست عروض تجارة)

**الخطوة 8:** يُصفى النقد — يُستبعد النقد المعادل (شبه النقد) كسندات الخزانة والودائع الاستثمارية.

**الخطوة 9:** يُصفى المخزون — يُستبعد البضاعة الكاسدة (ما انقطع الطلب عنها في سوقها).

**الخطوة 10:** يُستخرج ربع العشر من صافي الأصول الزكوية.

```
┌─────────────────────────────────────────────────────────────┐
│        حساب الزكاة — طريقة الموجودات الشرعية (DEFAULT)      │
│        Zakat — Sharia Assets Method                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  الأصول الزكوية (تجب فيها الزكاة):                          │
│  ──────────────────────────────────                         │
│  النقد والأرصدة البنكية           Cash & Bank Balances      │
│  (-) النقد المعادل/ودائع استثمارية  Less: Quasi-Cash         │
│  = صافي النقد                     = Net Cash                │
│                                                             │
│  عروض التجارة (مخزون للبيع)       Trade Goods (Inventory)   │
│  (-) بضاعة كاسدة/راكدة            Less: Stale/Obsolete      │
│  = صافي عروض التجارة              = Net Trade Goods          │
│                                                             │
│  ══════════════════════════════════                          │
│  الوعاء الزكوي = صافي النقد + صافي عروض التجارة              │
│  Zakat Base = Net Cash + Net Trade Goods                    │
│                                                             │
│  الأصول المستبعدة (لا زكاة فيها):                            │
│  ──────────────────────────────────                          │
│  ❌ المدينون — ملك ناقص                                     │
│  ❌ الاستثمارات — ملك ناقص                                   │
│  ❌ الأصول الثابتة — استهلاكية                               │
│  ❌ الأصول المعنوية — غير ملموسة                             │
│  ❌ مشاريع تحت الإنشاء — مال تحويلي لم يكتمل                │
│                                                             │
│  الزكاة = الوعاء × 2.5% (هجري) أو 2.5775% (ميلادي)        │
└─────────────────────────────────────────────────────────────┘
```

### 🟡 الطريقة الثالثة: الربح المُعدَّل — Adjusted Profit Method

```
┌─────────────────────────────────────────────────────────────┐
│              حساب الربح المُعدَّل                            │
│          Adjusted Profit Calculation                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  صافي الربح المحاسبي           Net Accounting Profit        │
│  = الإيرادات - المصروفات       = Revenue - Expenses         │
│                                                             │
│  (+) مصروفات غير جائزة الحسم:                               │
│      • مصروف الاستهلاك/الإهلاك  Depreciation                │
│      • مصروف المخصصات          Provision Expenses            │
│      • الغرامات والجزاءات      Penalties & Fines             │
│      • التبرعات فوق الحد       Excess Donations              │
│      • مصروف الضيافة الزائد    Excess Entertainment           │
│  ─────────────────────────────────────────────               │
│  = إجمالي الإضافات             Total Add-backs              │
│                                                             │
│  الربح المُعدَّل = صافي الربح + الإضافات                    │
│  Adjusted Profit = Net Profit + Add-backs                   │
│                                                             │
│  ══════════════════════════════════════                      │
│  الزكاة = الربح المُعدَّل × 2.5%  (هجري)                   │
│  الزكاة = الربح المُعدَّل × 2.5775% (ميلادي)               │
│  Zakat = Adjusted Profit × Rate                             │
└─────────────────────────────────────────────────────────────┘
```

#### مصروفات غير جائزة الحسم — Non-Deductible Expenses Mapping

| البند | كيف يتعرف عليه النظام | السبب الشرعي |
|-------|----------------------|-------------|
| الاستهلاك/الإهلاك | حسابات مصروف تحتوي "استهلاك" أو "إهلاك" أو `depreciation` | لأنه مصروف تقديري وليس نقدي فعلي |
| المخصصات | حسابات مصروف تحتوي "مخصص" أو `provision` | لأنها التزامات مقدرة لم تتحقق بعد |
| الغرامات | حسابات مصروف تحتوي "غرام" أو "جزاء" أو `penalty` | لأنها ناتجة عن مخالفات ولا تُخصم |

## 5. ملاحظات شرعية مهمة — Important Sharia Notes

### ✅ ما يدخل في الوعاء الزكوي (يُزكّى):
1. **النقد والأرصدة البنكية** — Cash & Bank Balances (الإجماع)
2. **عروض التجارة (المخزون المعد للبيع)** — Trade Goods/Inventory for sale (اتفاق الجماهير)
3. **المدينون المرجوون** — Collectible Receivables (جمهور الفقهاء + AAOIFI المعيار 35)
   - الشرط: أن يكون المدين قادراً على السداد وغير مماطل (مرجو التحصيل)
   - يُستبعد: الديون المشكوك فيها والمعدومة
4. **استثمارات المضاربة** — Trading Investments (قصيرة الأجل المعدة للبيع والشراء)

### ➖ ما يُخصم من الوعاء:
1. **الالتزامات المتداولة** — Current Liabilities (الديون الحالة قصيرة الأجل)
   - الدليل: تُنقص من الملك التام للأصول الزكوية (جمهور الفقهاء + AAOIFI)

### ❌ ما لا يدخل في الوعاء الزكوي (مستبعد):
1. **الأصول الثابتة** — Fixed Assets (استهلاكية — ليست نقوداً ولا عروض تجارة)
2. **الأصول المعنوية** — Intangible Assets (رخص، علامات تجارية، شهرة)
3. **تحت الإنشاء/التصنيع** — WIP/Under Construction (مال تحويلي لم يكتمل)
4. **المواد الخام وقطع الغيار** — Raw Materials/Spare Parts (ليست معدة للبيع)
5. **النقد المعادل (شبه النقد)** — Quasi-Cash (سندات خزانة، ودائع استثمارية)
6. **البضاعة الكاسدة** — Stale/Obsolete Inventory (انقطع الطلب عنها)
7. **استثمارات طويلة الأجل** — Long-term Investments
8. **الديون المشكوك فيها والمعدومة** — Doubtful/Bad Debts

### ⚠️ شروط وجوب الزكاة:
1. **بلوغ النصاب** — Reaching Nisab (85g gold equivalent)
2. **حولان الحول** — Completion of one year (Hijri)
3. **الملك التام** — Full ownership
4. **النماء** — Growth potential

### 📚 المراجع:
- **ZATCA** — هيئة الزكاة والضريبة والجمارك (السعودية)
- **AAOIFI** — معايير المحاسبة الإسلامية (المعيار 35 — الزكاة)
- **بيت الزكاة الكويتي** — دليل حساب زكاة الشركات

## 7. مثال عملي — Worked Example

### شركة افتراضية — بطريقة صافي الأصول المتداولة (AAOIFI):

```
═══ الأصول الزكوية (إضافات) ═══

النقد والأرصدة البنكية:               940,000 ريال
(-) النقد المعادل:                              0 ريال
= صافي النقد:                          940,000 ريال

عروض التجارة (مخزون للبيع):            120,000 ريال
(-) بضاعة كاسدة:                              0 ريال
= صافي عروض التجارة:                   120,000 ريال

المدينون المرجوون:                      200,000 ريال
استثمارات المضاربة:                     50,000 ريال
──────────────────────────────────
إجمالي الإضافات:                     1,310,000 ريال

═══ الحسميات ═══

الالتزامات المتداولة (ديون حالة):         150,000 ريال
──────────────────────────────────
الوعاء الزكوي = 1,310,000 - 150,000 = 1,160,000 ريال

أصول غير زكوية (للعلم — لا تدخل الحساب):
──────────────────────────────────
الأصول الثابتة:                        500,000 ريال
استثمارات طويلة الأجل:                100,000 ريال
أصول معنوية:                            50,000 ريال

═══ حساب الزكاة ═══
الزكاة (هجري):   1,160,000 × 2.5%    = 29,000.00 ريال
الزكاة (ميلادي): 1,160,000 × 2.5775% = 29,899.00 ريال
```

### نفس الشركة — بطريقة ZATCA (الافتراضية — للمقارنة):

```
رأس المال + احتياطيات + أرباح مبقاة:  1,000,000 ريال
الالتزامات طويلة الأجل:                300,000 ريال
المخصصات:                               80,000 ريال
صافي الربح:                            250,000 ريال
──────────────────────────────────
إجمالي الإضافات:                     1,630,000 ريال

(-) الأصول الثابتة:                    500,000 ريال
(-) استثمارات طويلة الأجل:             100,000 ريال
──────────────────────────────────
الوعاء = 1,630,000 - 600,000 = 1,030,000 ريال

الزكاة (هجري) = 1,030,000 × 2.5% = 25,750.00 ريال
```

## 8. التحقق من الصحة — Validation Checklist

للمراجع الشرعي، يرجى التحقق من:

| # | البند | الحالة | ملاحظة |
|---|-------|--------|--------|
| 1 | طريقة ZATCA (صافي الملكية) هي الافتراضية | ✅ | المعتمدة نظامياً في السعودية |
| 2 | طريقة AAOIFI تُدخل المدينون المرجوون | ✅ | وفقاً لجمهور الفقهاء |
| 3 | استبعاد المشكوك والمعدوم من المدينون | ✅ | الشرط: مرجو التحصيل |
| 4 | خصم الالتزامات المتداولة (الديون الحالة) | ✅ | جمهور الفقهاء + AAOIFI |
| 5 | إدراج استثمارات المضاربة (قصيرة الأجل) | ✅ | تُعامل كعروض تجارة |
| 6 | استبعاد الأصول الثابتة | ✅ | استهلاكية — لا زكاة فيها |
| 7 | استبعاد النقد المعادل من النقد | ✅ | ودائع استثمارية وسندات خزانة |
| 8 | استبعاد البضاعة الكاسدة | ✅ | انقطع الطلب عنها |
| 9 | النسبة 2.5% للسنة الهجرية | ✅ | مطابق للإجماع |
| 10 | تعديل النسبة 2.5775% للميلادية | ✅ | 2.5 × 365/354 |
| 11 | القيد المحاسبي مطابق | ✅ | Dr: مصروف → Cr: مستحق |
| 12 | التوافق مع معايير AAOIFI | ✅ | المعيار 35 |
