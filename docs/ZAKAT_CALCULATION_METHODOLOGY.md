## 1. الأساس الشرعي — Sharia Basis

| المصدر | النص |
|--------|------|
| القرآن الكريم | ﴿خُذْ مِنْ أَمْوَالِهِمْ صَدَقَةً تُطَهِّرُهُمْ وَتُزَكِّيهِم بِهَا﴾ — التوبة: 103 |
| الحديث الشريف | "فأعلمهم أن الله افترض عليهم صدقة تؤخذ من أغنيائهم فترد على فقرائهم" — متفق عليه |
| نصاب الذهب | 85 غرام ذهب عيار 24 |
| نصاب الفضة | 595 غرام فضة |
| النسبة | **2.5%** للسنة الهجرية (354 يوم) |
| | **2.5775%** للسنة الميلادية (365 يوم) = 2.5 × 365/354 |

## 2. طرق الحساب المدعومة — Supported Methods

### الطريقة الأولى: صافي الأصول (الوعاء الزكوي) — Net Assets Method

> هذه الطريقة المعتمدة من **هيئة الزكاة والضريبة والجمارك (ZATCA)** في المملكة العربية السعودية

```
┌─────────────────────────────────────────────────────────────┐
│                  حساب الوعاء الزكوي                         │
│            Zakat Base Calculation                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  (+) رأس المال المدفوع          Paid-up Capital             │
│  (+) الاحتياطيات                Reserves                    │
│  (+) الأرباح المبقاة           Retained Earnings            │
│  (+) المخصصات                  Provisions                   │
│  (+) القروض طويلة الأجل        Long-term Loans              │
│  (+) صافي الربح (إذا موجب)     Net Profit (if positive)     │
│  ─────────────────────────────────────────────               │
│  = إجمالي الإضافات             Total Additions              │
│                                                             │
│  (-) الأصول الثابتة             Fixed Assets (net)          │
│  (-) الاستثمارات طويلة الأجل    Long-term Investments       │
│  (-) خسائر ما قبل التشغيل      Pre-operating Losses        │
│  ─────────────────────────────────────────────               │
│  = إجمالي الحسميات             Total Deductions             │
│                                                             │
│  الوعاء الزكوي = الإضافات - الحسميات                        │
│  Zakat Base = Additions - Deductions                        │
│                                                             │
│  ══════════════════════════════════════                      │
│  الزكاة = الوعاء × 2.5%  (هجري)                            │
│  الزكاة = الوعاء × 2.5775% (ميلادي)                        │
│  Zakat = Base × Rate                                        │
└─────────────────────────────────────────────────────────────┘
```

#### ربط الحسابات في النظام — Account Mapping

| البند | رمز الحساب في النظام | نوع الحساب |
|-------|---------------------|------------|
| رأس المال | حسابات نوع `equity` | حقوق ملكية |
| الاحتياطيات | حسابات نوع `equity` تحتوي "احتياط" | حقوق ملكية |
| الأرباح المبقاة | حسابات نوع `equity` تحتوي "أرباح مبقاة" | حقوق ملكية |
| المخصصات | حسابات تحتوي "مخصص" (التزامات/حقوق ملكية) | التزامات |
| قروض طويلة الأجل | حسابات بادئة `22xx` أو تحتوي "طويل الأجل" | التزامات |
| صافي الربح | إيرادات (`revenue`) - مصروفات (`expense` + `cogs`) | محسوب |
| أصول ثابتة | حسابات بادئة `12xx` أو `15xx` أو تحتوي "أصول ثابتة" | أصول |
| استثمارات طويلة | حسابات بادئة `13xx` أو تحتوي "استثمار" | أصول |

### الطريقة الثانية: الربح المُعدَّل — Adjusted Profit Method

```
┌─────────────────────────────────────────────────────────────┐
│              حساب الربح المُعدَّل                            │
│          Adjusted Profit Calculation                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  صافي الربح المحاسبي           Net Accounting Profit        │
│  = الإيرادات - المصروفات       = Revenue - Expenses         │
│                                                             │
│  (+) مصروفات غير جائزة الحسم:                               │
│      • مصروف الاستهلاك/الإهلاك  Depreciation                │
│      • مصروف المخصصات          Provision Expenses            │
│      • الغرامات والجزاءات      Penalties & Fines             │
│      • التبرعات فوق الحد       Excess Donations              │
│      • مصروف الضيافة الزائد    Excess Entertainment           │
│  ─────────────────────────────────────────────               │
│  = إجمالي الإضافات             Total Add-backs              │
│                                                             │
│  الربح المُعدَّل = صافي الربح + الإضافات                    │
│  Adjusted Profit = Net Profit + Add-backs                   │
│                                                             │
│  ══════════════════════════════════════                      │
│  الزكاة = الربح المُعدَّل × 2.5%  (هجري)                   │
│  الزكاة = الربح المُعدَّل × 2.5775% (ميلادي)               │
│  Zakat = Adjusted Profit × Rate                             │
└─────────────────────────────────────────────────────────────┘
```

#### مصروفات غير جائزة الحسم — Non-Deductible Expenses Mapping

| البند | كيف يتعرف عليه النظام | السبب الشرعي |
|-------|----------------------|-------------|
| الاستهلاك/الإهلاك | حسابات مصروف تحتوي "استهلاك" أو "إهلاك" أو `depreciation` | لأنه مصروف تقديري وليس نقدي فعلي |
| المخصصات | حسابات مصروف تحتوي "مخصص" أو `provision` | لأنها التزامات مقدرة لم تتحقق بعد |
| الغرامات | حسابات مصروف تحتوي "غرام" أو "جزاء" أو `penalty` | لأنها ناتجة عن مخالفات ولا تُخصم |

## 5. ملاحظات شرعية مهمة — Important Sharia Notes

### ✅ ما يدخل في الوعاء الزكوي (يُزكّى):
1. **النقد والأرصدة البنكية** — Cash & Bank Balances
2. **الذمم المدينة القابلة للتحصيل** — Collectible Receivables
3. **المخزون بسعر السوق أو التكلفة (أيهما أقل)** — Inventory at lower of cost/market
4. **الاستثمارات قصيرة الأجل** — Short-term Investments
5. **أرباح السنة غير الموزعة** — Undistributed Current Year Profit

### ❌ ما لا يدخل في الوعاء الزكوي (يُحسم):
1. **الأصول الثابتة المستخدمة في النشاط** — Fixed Assets used in operations
2. **الاستثمارات العقارية طويلة الأجل** — Long-term Real Estate Investments
3. **مصاريف التأسيس** — Pre-operating Costs
4. **الخسائر المرحّلة** — Carried-forward Losses

### ⚠️ شروط وجوب الزكاة:
1. **بلوغ النصاب** — Reaching Nisab (85g gold equivalent)
2. **حولان الحول** — Completion of one year (Hijri)
3. **الملك التام** — Full ownership
4. **النماء** — Growth potential

## 7. مثال عملي — Worked Example

### شركة افتراضية — بطريقة صافي الأصول:

```
رأس المال المدفوع:                  1,000,000 ريال
الاحتياطيات:                          200,000 ريال
الأرباح المبقاة:                      150,000 ريال
المخصصات:                              80,000 ريال
القروض طويلة الأجل:                   300,000 ريال
صافي ربح السنة:                       250,000 ريال
────────────────────────────────────────────────
إجمالي الإضافات:                    1,980,000 ريال

(-) الأصول الثابتة (صافي):            600,000 ريال
(-) استثمارات طويلة الأجل:            100,000 ريال
────────────────────────────────────────────────
إجمالي الحسميات:                      700,000 ريال

الوعاء الزكوي = 1,980,000 - 700,000 = 1,280,000 ريال

الزكاة (هجري):   1,280,000 × 2.5%    = 32,000.00 ريال
الزكاة (ميلادي): 1,280,000 × 2.5775% = 32,992.00 ريال
```

## 8. التحقق من الصحة — Validation Checklist

للمراجع الشرعي، يرجى التحقق من:

| # | البند | الحالة | ملاحظة |
|---|-------|--------|--------|
| 1 | النسبة 2.5% للسنة الهجرية | ✅ | مطابق للإجماع |
| 2 | تعديل النسبة 2.5775% للميلادية | ✅ | حسب فتوى هيئة كبار العلماء |
| 3 | حسم الأصول الثابتة | ✅ | لا زكاة في أصول الاستعمال |
| 4 | إضافة المخصصات | ✅ | لأنها مال مرصود لم يُنفق |
| 5 | إضافة القروض طويلة الأجل | ✅ | حسب رأي ZATCA |
| 6 | حساب صافي الربح | ✅ | يُضاف إذا كان موجباً |
| 7 | الإضافات غير المحسومة للربح المعدل | ✅ | استهلاك + مخصصات + غرامات |
| 8 | القيد المحاسبي مطابق | ✅ | Dr: مصروف → Cr: مستحق |
