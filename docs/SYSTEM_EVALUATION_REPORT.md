## 4.4 التنفيذ التقني المطلوب

### الخطوة 1: جدول القوالب في قاعدة البيانات

```sql
-- قالب النشاط
CREATE TABLE industry_templates (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100) NOT NULL,        -- 'retail', 'restaurant', 'manufacturing'
    name_ar VARCHAR(100) NOT NULL,     -- 'محلات تجارية', 'مطاعم', 'تصنيع'
    icon VARCHAR(10),                  -- '🏪', '🍔', '🏭'
    description_ar TEXT,
    enabled_modules JSONB NOT NULL,     -- ["accounting","sales","inventory","pos"]
    hidden_modules JSONB,              -- ["manufacturing","projects"]
    default_chart_of_accounts JSONB,   -- شجرة حسابات افتراضية
    default_settings JSONB,            -- إعدادات افتراضية
    default_dashboard_widgets JSONB,   -- widgets لوحة التحكم
    default_reports JSONB              -- تقارير مخصصة
);

-- ربط الشركة بالقالب
ALTER TABLE companies ADD COLUMN template_id INTEGER REFERENCES industry_templates(id);
ALTER TABLE companies ADD COLUMN enabled_modules JSONB; -- تخصيص إضافي
```

### الخطوة 2: تعديل `Sidebar.jsx`

// بدلاً من عرض كل الوحدات

const menuItems = allMenuItems.filter(item =>

);
